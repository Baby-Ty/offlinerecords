Welcome

Welcome to the North Country Associates Disaster Recovery Interface. This interface is ment to be used during any pirode of time when your facility has lost network connectivity and can no longer access MARS, TARS or Face Sheets regularly.

Accessing Records

On this page you will find four buttons, one for MARS, one for TARS, one for orders, and one for Face Sheets. All records have been updated within the last hour that the facility had connectivity. To access a record close this popup, then click the button associated with the record type that you wish to view. This will will open the record selection menu where you can select the spacific record that you need.
